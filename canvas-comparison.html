Canvas Comparison - 2D, WebGL, Offscreen

Four canvases rendering the same DLP test data using different contexts:

  • Canvas 1: Standard 2D context (same as canvas.html)
  • Canvas 2: WebGL context (renders via texture)
  • Canvas 3: Offscreen canvas with 2D (renders same data)
  • Canvas 4: Offscreen canvas with WebGL (renders same data)

Testing WebGL fallback with Chrome DevTools MCP

To test the "2D (fallback)" badges, disable WebGL and reload. Chrome DevTools MCP supports --chromeArg to pass Chrome flags: 

{
  "mcpServers": {
    "chrome-devtools": {
      "command": "npx",
      "args": [
        "-y",
        "chrome-devtools-mcp@latest",
        "--chromeArg=--disable-webgl"
      ]
    }
  }
}

Or connect to a Chrome instance started manually with WebGL disabled (macOS): 

/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222 --user-data-dir=/tmp/chrome-profile-stable --disable-webgl

Then use --browser-url=http://127.0.0.1:9222 in your MCP config.
